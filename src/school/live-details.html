课程简介

  • 本讲重点讲解攻克"高考语文古诗词鉴赏·炼字"难题的绝招，牛哥独创"12字诀、四步法",教你轻松玩转"炼字"小宇宙。本讲重点讲解攻克"高考语文古诗词鉴赏·炼字"难题的绝招，牛哥独创"12字诀、四步法",教你轻松玩转"炼字"小宇宙。

课程亮点

  • 1、为一轮复习提供古诗词鉴赏得分秘籍；
  • 2、四步炼词法、两句意境法等等牛哥独创的解题技巧
  • 3、让你的古诗词鉴赏从此"柳暗花明、绝处逢春"。

课程VCR

课程目录

  • 第一节 应用文的谋篇之物品介绍 ( 时长：20分54秒 )
  • 第二节 牛哥独创"12字诀、四步法" ( 时长：20分54秒 )
  • 第三节 应用文的谋篇之物品介绍 ( 时长：20分54秒 )
